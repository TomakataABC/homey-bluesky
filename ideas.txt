Bluesky Homey App Ideas

This file summarizes realistic feature ideas based on the official Bluesky / ATProto documentation.

Core actions the app could support

- Create a post
- Create a reply
- Quote a post
- Repost and undo repost
- Like and unlike a post
- Delete own post
- Post with images
- Post with video
- Post with link embeds
- Post with mentions and rich-text facets
- Post with reply restrictions / thread gates
- Follow and unfollow users
- Block and unblock users
- Mute and unmute users
- Create and manage lists
- Add and remove users from lists
- Mute and block lists
- Update display name, bio, avatar, and banner
- Change notification preferences
- Submit moderation reports
- Potentially send direct messages via official chat APIs

Data the app could report

Account/profile data
- Handle
- DID
- Display name
- Bio
- Avatar or banner presence
- Follower count
- Following count
- Post count

Notifications and activity
- Unread notification count
- Notification list
- Mentions
- Likes on your posts
- Reposts of your posts
- Replies to your posts
- New followers

Feed and post data
- Latest post text
- Latest post URI
- Latest post timestamp
- Latest post like, repost, reply, and quote counts
- Timeline latest item
- Custom feed latest item
- Author feed latest item

Relationship and moderation state
- Whether you follow a target user
- Whether a target user follows you
- Whether a user is muted
- Whether a user is blocked
- Whether a list is muted
- Whether a list is blocked

List data
- Lists you own
- Lists a user is in
- List member count
- Latest activity in a list feed

Possible Homey Flow trigger cards

- New mention
- New follower
- New reply to my post
- New like on my post
- New repost of my post
- New quote of my post
- Unread notifications became non-zero
- Watched author posted
- Custom feed has a new post
- Timeline has a new post matching criteria
- Watched thread received a reply
- Follower count changed
- Following count changed
- Profile changed
- Post engagement crossed a threshold
- Mute, block, or list state changed

Possible Homey Flow condition cards

- Has unread notifications
- Is user followed
- Is user muted
- Is user blocked
- Is list muted
- Is list blocked
- Follower count is above X
- Latest post contains text, link, or image
- Specific post has more than X likes, reposts, or replies
- Profile handle or display name matches a value

Possible Homey Flow action cards

- Create post
- Reply to post
- Quote post
- Like / unlike post
- Repost / undo repost
- Follow / unfollow user
- Mute / unmute user
- Block / unblock user
- Add user to list
- Remove user from list
- Update profile details
- Change notification settings
- Send DM if chat support is added

Possible Homey widgets

- Profile summary widget with avatar, handle, display name, and counts
- Unread notifications widget with count and latest notifications
- Latest post widget with text, timestamp, and engagement counts
- Engagement widget for a selected post
- Watched feed widget for latest posts from an author, list, or custom feed
- Mentions widget
- Thread or reply tracker widget
- List activity widget
- Media spotlight widget
- DM or conversation widget

Important technical notes

- Most read and event features will likely require polling.
- Bluesky has a firehose for public network updates, but it is not the simplest source for per-account Homey automations.
- Notifications, mutes, some preferences, and chat are authenticated or private.
- Media support adds extra complexity.
- Moderation labels and visibility rules must be respected in widgets and data displays.

Suggested roadmap

Phase 1
- Create post
- Reply to post
- Like and repost
- New mention trigger
- New follower trigger
- Unread notification count
- Profile summary widget

Phase 2
- Post with images
- Follow and unfollow
- Watched author posted trigger
- Post engagement triggers
- Latest post widget
- Notifications widget

Phase 3
- List support
- Profile update actions
- Moderation actions
- Custom feed monitoring
- Thread tracking
- DM features if they prove useful

Summary

Bluesky is capable of supporting a much broader Homey integration than posting alone. The best areas for expansion are notifications, engagement tracking, profile and graph status, feed monitoring, and a small set of focused widgets.
