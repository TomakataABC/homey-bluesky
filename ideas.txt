Bluesky Homey App Ideas

This file summarizes realistic feature ideas based on the official Bluesky / ATProto documentation.

Core actions the app could support

- [x] Create a reply
- [x] Quote a post
- [x] Repost and undo repost
- [x] Like and unlike a post
- [x] Delete own post
- Post with images
- Post with video
- Post with link embeds
- [x] Post with mentions and rich-text facets
- [x] Follow and unfollow users
- [x] Block and unblock users
- [x] Mute and unmute users
- Update display name, bio, avatar, and banner

Data the app could report

Account/profile data
- [x] Handle
- [x] DID
- [x] Display name
- Bio
- Avatar or banner presence
- [x] Follower count
- [x] Following count
- [x] Post count

Notifications and activity
- [x] Unread notification count
- [x] Notification list
- [x] Mentions
- [x] Likes on your posts
- [x] Reposts of your posts
- [x] Replies to your posts
- [x] New followers

Feed and post data
- [x] Latest post text
- [x] Latest post URI
- [x] Latest post timestamp
- [x] Latest post like, repost, reply, and quote counts
- Timeline latest item
- Custom feed latest item
- Author feed latest item

Relationship and moderation state
- [x] Whether you follow a target user
- Whether a target user follows you
- Whether a user is muted
- Whether a user is blocked

Possible Homey Flow trigger cards

- [x] New mention
- [x] New follower
- [x] New reply to my post
- [x] New like on my post
- [x] New repost of my post
- [x] New quote of my post
- [x] Unread notifications became non-zero
- Watched author posted
- Custom feed has a new post
- Timeline has a new post matching criteria
- Watched thread received a reply
- Follower count changed
- Following count changed
- Profile changed
- Post engagement crossed a threshold

Possible Homey Flow condition cards

- Has unread notifications
- [x] Is user followed
- [x] Is user muted
- [x] Is user blocked
- [x] Is followed by user
- Follower count of specific user is above X
- Specific post has more than X likes, reposts, or replies

Possible Homey Flow action cards

- [x] Create post
- Reply to post
- [x] Quote post
- [x] Like / unlike post
- [x] Repost / undo repost
- [x] Follow / unfollow user
- [x] Mute / unmute user
- [x] Block / unblock user
- Update profile details
- Change notification settings
- Send DM if chat support is added

Possible Homey widgets

- [ ] Profile summary widget with avatar, handle, display name, and counts (implemented but disabled for now)
- [ ] Unread notifications widget with count and latest notifications (implemented but disabled for now)
- [ ] Latest post widget with text, timestamp, and engagement counts (implemented but disabled for now)
- Engagement widget for a selected post
- Mentions widget
- Thread or reply tracker widget
- Media spotlight widget
